# Will-Tecnologia: Modernização de Sistema de Dados
> Projeto de Administração e Gerenciamento de Banco de Dados focado em Políticas de Retenção e Automação Industrial.

## 📌 Sobre o Projeto
Este projeto foi desenvolvido como parte da atividade prática do curso de Sistemas de Armazenamento de Dados para Automação (SENAI). O objetivo principal é a modernização do sistema de armazenamento da empresa Will-Tecnologia, estruturando um banco de dados robusto que suporte desde cadastros de clientes até dados de sensores industriais.

- O foco da missão é aplicar conceitos de:
- Modelagem de Dados Relacional.
- Políticas de Retenção de Dados (Data Retention).
- Segurança da Informação.
- Automação de Coleta de Dados.

## 🏗️ Estrutura do Banco de Dados
O sistema foi desenhado para controlar seis pilares principais da empresa:
1. **Clientes**: Cadastro completo e atualizado.
2. **Vendas**: Registro histórico de transações comerciais.
3. **Logs de Acesso**: Rastreabilidade de quem e quando acessou o sistema.
4. **Histórico de Clientes**: Dados de clientes antigos para conformidade e análise.
5. **Backups**: Registro e controle de integridade das cópias de segurança.
6. **Sensores Industriais**: Armazenamento de dados provenientes da automação fabril.

## 🛠️ Ferramentas Usadas
- SGBD: Microsoft SQL Server 2022
- Interface: SQL Server Management Studio (SSMS)
